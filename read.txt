To run it is needed to change the values in the esp-32 since the SSID Password and ip are hard coded.

idf.py build
idf.py  -p flash /dev/ttyUSB0
python3 -m http.server 8000
python3 webServer.py

double check if the esp and host laptop are in the same network
check esp SSID and password for acess to the network
the dashboard will be hosted in localhost:8000


Commands for importing and allowing serial port
sudo chmod 0777 /dev/ttyUSB0
. $HOME/esp/esp-idf/export.sh
